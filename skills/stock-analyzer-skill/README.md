# Stock Analyzer Skill 使用指南

## 概述

`stock-analyzer-skill` 是一个全球股票综合分析工具，支持 A股、港股、美股等东方财富覆盖的所有市场。

## 功能特性

- 📊 **多市场支持**：A股（沪市/深市/北交所）、港股、美股
- 📈 **实时行情**：获取实时股价、涨跌幅、成交量等数据
- 💰 **基本面分析**：PE、PB、市值、换手率等估值指标
- 📰 **新闻面分析**：公司公告、新闻、研报等信息
- 💵 **资金面分析**：主力资金流向、北向资金等
- 🎯 **投资建议**：综合评分、买卖价位建议
- 📄 **可视化报告**：生成 HTML 格式的分析报告

## 在 GoClaw 中使用

### 1. 通过 HTTP API 使用

启动 GoClaw daemon 后，可以通过 HTTP API 调用股票分析功能：

```bash
# 启动 GoClaw daemon
./bin/goclaw daemon

# 在另一个终端发送请求
curl -X POST http://localhost:4096/agent \
  -H "Content-Type: application/json" \
  -d '{
    "message": "帮我分析一下贵州茅台的股票"
  }'
```

### 2. 支持的查询方式

#### 按股票名称查询
```
分析贵州茅台
分析腾讯控股
分析苹果AAPL
分析美光MU
```

#### 按股票代码查询
```
分析 600519
分析 00700
分析 AAPL
分析 MU
```

#### 指定市场类型
```
分析 600519，市场是沪市
分析 00700，市场是港股
分析 AAPL，市场是美股
```

### 3. 触发关键词

以下关键词会自动触发股票分析技能：

- **通用**：分析股票、股票推荐、股票买卖点、股票研究
- **A股**：A股分析、沪深股票、创业板、科创板、北交所
- **港股**：港股分析、恒生指数、腾讯、美团、小米、比亚迪股份
- **美股**：美股分析、纳斯达克、标普500、道琼斯、中概股、苹果AAPL、特斯拉TSLA、英伟达NVDA、美光MU、谷歌GOOGL、微软MSFT、亚马逊AMZN、Meta、阿里巴巴BABA、拼多多PDD、京东JD

## 工作原理

1. **用户请求**：用户通过 HTTP API 发送股票分析请求
2. **工具调用**：GoClaw 调用 `stock_analyze` 工具
3. **技能执行**：Node.js 脚本调用 Python 脚本获取数据
4. **数据获取**：从东方财富 API 获取实时行情数据
5. **分析报告**：生成综合分析报告（Markdown 格式）
6. **返回结果**：将分析报告返回给用户

## 数据来源

- **主要数据源**：东方财富网 (https://www.eastmoney.com)
- **API 接口**：东方财富实时行情 API
- **数据更新**：实时数据，延迟约 1-2 秒

## 注意事项

1. **登录建议**：建议登录东方财富网账号以获取完整数据
2. **请求限制**：东方财富对高频请求有限流，请勿短时间内频繁查询
3. **数据准确性**：所有数据仅供参考，不构成投资建议
4. **市场时间**：仅在交易时间内能获取实时数据
5. **Python 依赖**：需要安装 Python 3 和相关依赖包

## 故障排除

### 问题 1：Python 脚本执行失败
- 检查 Python 是否安装：`python3 --version`
- 安装依赖：`pip install -r scripts/requirements.txt`

### 问题 2：无法获取股票数据
- 检查网络连接
- 确认股票代码是否正确
- 尝试登录东方财富网账号

### 问题 3：GoClaw 构建失败
- 确保已正确添加 `NewStockAnalyzerTool` 到 `cmd/main.go`
- 运行 `go mod tidy` 更新依赖
